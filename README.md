# ESP jQuery Mobile Interface
This example here is using a Captive Portal, for an easy connection with the single webpage application. Try it, you have only select the SSID "SleepUinoPreDev" in the WLAN list of available network on your mobile device. Thatâ€™s it, now the single webpage application will start on your mobile device. 

**A simple interface connection for all WLAN devices, independently of the used os platform.**

If you have an ESP8266 or ESP32 you can find final binaries of this example in the [ESPEasyFlasher2.0 Release Assets](https://github.com/hredan/ESPEASYFLASHER_2.0/releases/latest).
